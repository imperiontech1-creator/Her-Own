# HER OWN — Logo system

Transparent-background SVGs. Color modes: black (#0B0A0A), gold (#C9A227), bone (#F3EDE7), white (#FFFFFF), deep plum (#3A0F1B).

- **wordmark-{color}.svg** — HER OWN wordmark only
- **emblem-{color}.svg** — emblem only (symmetrical mark in thin ring)
- **lockup-{color}.svg** — emblem + vertical divider + wordmark

Used in app: navbar uses `wordmark-black.svg` on light backgrounds.
